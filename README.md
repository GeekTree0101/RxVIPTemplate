# Karrot Clean Swift Template

# Install
To install the Karrot Clean Swift Xcode templates, run:

> make install

To uninstall the Karrot Clean Swift Xcode templates, run:

> make uninstall

# Requirements
- Texture(AsyncDisplayKit)
